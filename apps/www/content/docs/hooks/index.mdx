title: Hooks

---

Azemmur provides a collection of utility hooks that power the interactive components in the library. These hooks can be used independently to build custom components with similar functionality.

## Available Hooks

- [useFocusTrap](/docs/hooks/use-focus-trap) — Trap focus within a container for accessible modals and dialogs.
- [useIsMobile](/docs/hooks/use-is-mobile) — Detect mobile viewport with customizable breakpoint.
- [useKeyActions](/docs/hooks/use-key-actions) — Handle keyboard actions with customizable key bindings and modifiers.
- [useMergeRefs](/docs/hooks/use-merged-refs) — Merge multiple refs into one stable callback ref.
- [usePinchZoom](/docs/hooks/use-pinch-zoom) — Handle pinch-to-zoom gestures on touch devices.
