## API Reference

### Button

<TypeTable
  type={{
    hoverScale: {
      description: 'The scale of the button on hover.',
      type: 'number',
      required: false,
      default: 1.05,
    },
    tapScale: {
      description: 'The scale of the button on tap.',
      type: 'number',
      required: false,
      default: 0.95,
    },
    '...props': {
      description:
        'Additional props passed to the underlying motion-enabled button, such as event handlers, ARIA attributes, or native HTML props.',
      type: 'HTMLMotionProps<"button">',
      required: false,
    },
  }}
/>

> Note: This primitive respects the user's `prefers-reduced-motion` setting. When reduced motion is requested, hover/tap scale animations are disabled.
