## Usage

```tsx
// Uncontrolled usage
<Tabs tabs={['Overview', 'Features']} />;

// Controlled usage with change handler
const [active, setActive] = React.useState(0);

<Tabs
  tabs={['Overview', 'Features']}
  activeTabId={active}
  onTabChange={(index) => setActive(index)}
/>;
```

## Accessibility Notes

To ensure full accessibility, pair this component with matching content panels. The component emits `id` and `aria-controls` for triggers and expects panels to use those ids:

```tsx
<div
  role="tabpanel"
  id={`panel-${tabsId}-${activeTabId}`}
  aria-labelledby={`tab-${tabsId}-${activeTabId}`}
>
  {/* Content for the active tab */}
</div>
```

Keyboard support:

- ArrowRight / ArrowLeft (direction depends on `ltr`) to move between tabs.
- `Home` / `End` to jump to first/last tab.

Implementation notes:

- The `Tabs` component controls focus internally via `buttonRef` callbacks and maintains an internal active index unless `activeTabId` is provided (controlled mode).
- `aria-orientation` is set to `horizontal` and the container uses `role="tablist"`.
